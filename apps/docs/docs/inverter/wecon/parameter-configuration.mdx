Tài liệu này tổng hợp toàn bộ bảng mã tham số chức năng (Function Codes) của dòng biến tần **WECON VM Series** dựa trên sổ tay hướng dẫn chính thức phiên bản **V2.0 (20251203)**. 

Bảng bao gồm mã hàm chuẩn xác, tên thông số, dải giá trị cài đặt, giá trị mặc định nhà sản xuất, tính chất thay đổi và **địa chỉ RAM giao tiếp Modbus**.

---

## 1. Ký hiệu đặc tính thay đổi tham số (Properties)

* **`○`**: Giá trị tham số có thể thay đổi được trong cả 2 trạng thái **Dừng (Stop)** hoặc **Đang chạy (Running)**.
* **`●`**: Giá trị tham số chỉ có thể thay đổi khi biến tần **Dừng (Stop)**, không cho phép sửa khi đang chạy.
* **`◎`**: Giá trị chỉ đọc (Read-only / Detection record), không cho phép người dùng chỉnh sửa.

## 7. Nhóm C0: Nhóm thông số giám sát hoạt động (Monitoring Parameters)

Các tham số thuộc nhóm **C0** là tham số chỉ đọc (Read-only), dùng để kiểm tra trực tiếp trạng thái thực tế trên bàn phím hoặc qua mạng truyền thông Modbus:

| Mã hàm | Tên thông số giám sát | Đơn vị / Mô tả dải hiển thị | Địa chỉ Modbus RAM |
| :---: | --- | --- | :---: |
| **C0.00** | Running Frequency | Tần số ngõ ra thực tế (Độ phân giải: 0.01 Hz) | 1001H |
| **C0.01** | Set Frequency | Tần số cài đặt hiện tại (Độ phân giải: 0.01 Hz) | 1002H |
| **C0.02** | Output Current | Dòng điện đầu ra thực tế (Độ phân giải: 0.01 A) | 1005H |
| **C0.03** | Output Voltage | Điện áp đầu ra động cơ (Độ phân giải: 0.1 V) | 1004H |
| **C0.04** | Bus Voltage | Điện áp DC Bus thực tế (Độ phân giải: 0.1 V) | 1003H |
| **C0.05** | Module Temperature | Nhiệt độ khối IGBT (Độ phân giải: 0.1 °C) | 100DH |
| **C0.06** | DI Input Status | Trạng thái vật lý ngõ vào số DI1 ~ DI4 (Trọng số bit nhị phân) | 1008H |
| **C0.08** | AI Input Voltage | Điện áp tín hiệu Analog ngõ vào AI1 (Độ phân giải: 0.1 V) | 100AH |
| **C0.15** | Motor RPM | Tốc độ vòng quay động cơ (Độ phân giải: 1 RPM) | 100FH |
